## Phase 3: E-commerce & Systems (Weeks 8-16)
### "Make ordering easy"

#### 3.1 Online Ordering System

**Option A: Simple (Recommended to Start)**
- Square Online (free, takes ~2.9% + 30¬¢ per transaction)
- Syncs with Square POS if they get one
- Handles payments, order management
- Can set pickup times/dates

**Option B: Full E-commerce**
- Shopify ($29/month)
- More customization
- Better for scaling
- Built-in shipping options

**Recommendation:** Start with Square Online. Zero cost, professional checkout, easy to manage.

#### 3.2 Order Management
- Set up order notifications (email + text)
- Create production calendar
- Implement order cutoff times (e.g., order by Wednesday for Saturday pickup)
- Build simple inventory tracking

#### 3.3 Customer Database
- All orders automatically build customer list
- Track repeat customers
- Birthday/anniversary reminders
- "Haven't ordered in a while" win-back emails

## Phase 4: Scale & Ship (Months 4-6+)
### "Go National"

#### 4.1 Shipping Infrastructure

**Frozen Sourdough Model:**
Many successful bakeries ship frozen bread that customers bake at home:
- Bread is par-baked (80% done)
- Flash frozen
- Ships in insulated box with ice packs
- Customer finishes in home oven (10-15 min)
- Arrives "fresh from the oven"

**Shipping Partners:**
- ShipBob (fulfillment)
- Goldbelly (artisan food marketplace - they handle everything)
- Direct shipping via UPS/FedEx with insulated packaging

**Packaging Costs (estimated):**
| Item | Cost |
|------|------|
| Insulated box | $3-5 |
| Ice packs | $1-2 |
| Packaging materials | $1-2 |
| Shipping (2-day) | $15-25 |

**Minimum viable shipping price:** ~$35-40 for a loaf (includes product + shipping + margin)

#### 4.2 National Landing Pages

Build location pages for every major city:
- `/sourdough-bread-delivery-los-angeles`
- `/artisan-bread-shipped-new-york`
- `/best-sourdough-online-chicago`

**Scale:** 200-500 city pages targeting "sourdough delivery [city]" searches

#### 4.3 Subscription Model

**"Bread Club" Subscription:**
- Monthly delivery of fresh/frozen sourdough
- $45-65/month depending on frequency
- Predictable revenue
- Loyal customer base

**Example Tiers:**
| Tier | Contents | Price |
|------|----------|-------|
| Starter | 1 loaf/month | $45 |
| Baker | 2 loaves/month | $75 |
| Family | 4 loaves/month | $125 |
